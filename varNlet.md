#Varibale

###var


```
var msg = '안녕하세요, JavaScript';
var x 	= 10;
```

```=```는 오른쪽 내용을 변수에 대입하라는 명령 연산자임. 변수 msg에 '안녕하세요, JavaScript'라는 문자를, 변수 x에 10이라는 숫자를 각각 대입한다는 의미

#Variable with LET

###let

```
// Default 선언
let msg;

// Mutiple var 선언
let x, y;

// 초기값 설정
let greeting = 'Hello, World';

```

var/let과는 어떤 차이길래? <br />
1. 블록 스코프를 인식 <br />
2. 변수 중복을 허가하지 않음 <br />
- let
